## Levantar Proyecto üñ•Ô∏è

1. Clonar el repositorio.

2. Tener instalado `yarn`, sino instalarlo con:

```bash
npm install --global yarn
```

3. Crear una copia del `.env.template` y renombrarlo a `.env` y cambiar las variables de entorno.

```bash
STAGE=dev
PORT=3000 #default port

# Complete this fields with your own keys
POSTGRES_PASSWORD=YOUR_DB_PASS
POSTGRES_DATABASE=YOUR_DATABASE_NAME
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_USER=postgres #default user on postgresql
```

4. Instalar dependencias

```bash
yarn install
```

5. Crear la base de datos postgresql (Requisito: docker instalado)

```bash
docker compose up -d
```

6. Levantar el proyecto (en dev mode)

```bash
yarn start:dev
```
